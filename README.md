# GreetingKit

It is example project to test Carthage & Cocoa touch framework which uses swift, c and c++ files at the same time.

```swift
import UIKit
import GreetingKit

Hello("Hello")

HelloC("HaHaHaHa")

HelloCppWrapper.HelloCpp("Nice to meet you")
```
